# VRP-OptimizacionDeRutas
Este proyecto abordó el reto de optimizar las entregas de e-commerce ante un crecimiento proyectado del 12% en ventas en línea para 2024. Con técnicas de optimización estocástica, desarrollé un sistema que define rutas de vehículos y asignación de recursos para entregas en 344 ubicaciones de Monterrey.
